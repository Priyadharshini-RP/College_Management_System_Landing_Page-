# PDKVCET College Website

A responsive and interactive college website developed using **HTML5, CSS3, and JavaScript**. This project showcases a modern educational institution website featuring academic information, campus facilities, admissions, image gallery, and contact details through an engaging and user-friendly interface.

## Features

* Responsive design for desktop, tablet, and mobile devices
* Attractive hero section with call-to-action buttons
* About section with college overview
* Animated statistics counters
* Course filtering by category
* Interactive admissions table with sorting functionality
* Campus facilities showcase
* Image gallery with lightbox preview
* Contact form with client-side validation
* Sticky navigation bar with active section highlighting
* Smooth scrolling and responsive layout

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Font Awesome
* Google Fonts (Poppins)

## Project Structure

* `index.html` – Website structure
* `style.css` – Styling, layout, animations, and responsive design
* `script.js` – Interactive features, animations, filtering, sorting, gallery, and form validation

## Purpose

This project was developed to demonstrate front-end web development skills by creating a professional college website with modern UI design, responsive layouts, and interactive JavaScript functionality, providing an informative and engaging experience for prospective students and visitors.
